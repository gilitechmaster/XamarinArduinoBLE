아두이노 BLE 데이터 통신을 구현하는 전체 코드입니다.

Xabre Bluetooth LE plugin for Xamarin을 사용합니다.

권한요청 없이 상업적 이용
및 수정 재배포가 가능합니다.

많이 만들어주세요 ^^.
